# UI Mockup

This repository contains HTML mockups for the OmniBazaar user interface. These mockups are used for:

- User acceptance testing
- Interface design validation
- Demonstrating application functionality
- Simulating the appearance and workflow of the application

## Structure

The mockups include the following pages:

- **index.html** - Main index page with statistics and navigation
- **welcome-page.html** - Initial welcome/landing page
- **home-page.html** - User home dashboard
- **marketplace-page.html** - Main marketplace interface
- **marketplace-categories.html** - Category browsing interface
- **search-results.html** - Search results display
- **listing-detail.html** - Individual listing details
- **create-listing.html** - Create new marketplace listing
- **nft-mint-page.html** - NFT minting interface

## Assets

- **Globe-*.png** - OmniBazaar logo assets in various sizes

## Usage

Open `index.html` in a web browser to navigate through the mockup interface. All pages are cross-linked for easy navigation and testing of user workflows.

## Development

When creating new user interface pages in the main OmniBazaar TypeScript application, corresponding HTML mockups should be added to this directory to maintain consistency between the actual application and the testing interface.
